# Selenium JUnit Practice Project

## Description
This is a practice project to learn Selenium automation with Java and JUnit.  
It includes basic web automation scripts and simple test cases.

## Technologies Used
- Java
- Selenium WebDriver
- JUnit
- Maven

1. Clone the repository:
   ```bash
   git clone https://github.com/FablihaNahid/Selenium_Junit_Automation

Run the test cases using JUnit.
